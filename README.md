# rgb-gol-mojo

An RGB version of Conway's "Game of Live" in Mojo.

Red, green an blue cells life on a GOL grid and influence each other.

### Install
`pixi install`

### Start
`pixi run life`

Tap `<ENTER>` to start a new game, `<SPACE>` to pause/resume, or `<ESC>` to quit.
